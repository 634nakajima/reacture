Features

対面授業の「教室の盛り上がり」を創出する、リアルタイムインタラクション機能

絵文字リアクション

学生がスマホからワンタップでリアクションを送信。画面上にアニメーション表示されます。

効果音

リアクションに応じた効果音を再生。教室の盛り上がりを聴覚的にも演出します。

ニコニコ風コメント

学生からのコメントがスライド上を右から左へ流れるように表示されます。

アンケート

リアルタイム投票機能。結果はグラフで即座に可視化されます。

Q&A

学生が匿名で質問を投稿。いいね機能で注目度の高い質問が上位に表示されます。

デスクトップオーバーレイ

macOSネイティブアプリで、PowerPoint・Keynote等の上にリアクションを重ねて表示。

How it works

3ステップで教室をインタラクティブに

ルームを作成

PCまたはデスクトップアプリからルームを作成。4桁のルームコードが発行されます。

QRコードで共有

QRコードをプロジェクターに表示。学生はスマホでスキャンして即参加。

リアクション開始

学生がリアクション・コメント・質問を送信。教室が一体になります。

Architecture

モダンな技術スタックで構築

Web App

Next.js / React / Tailwind CSS

発表者画面・参加者画面をWebアプリとして提供。Vercelにデプロイ。

Desktop App

Electron / electron-builder

macOSネイティブアプリ。透明オーバーレイで任意のアプリ上にリアクションを表示。

Real-time Server

Socket.IO / Node.js

リアルタイム双方向通信でリアクション・コメント・投票を即時配信。Renderにデプロイ。
